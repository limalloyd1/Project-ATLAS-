Welcome to Project Argus:

This doc will serve as a brief introduction to the functionality and services that Argus provides. 

ARGUS stands for: Automated Rendering Guidance and Understanding System 

It is mainly used as a tool to assist troubleshooting and sanity checking OpenGL projects, but the skeleton is malleable enough to adjust to any data set. The goal is to allow for users to construct their own dynamic libraries to reference as you work on new projects.


###SCOPE###
The codebase will be made for hosting on a Rasberry Pi 4 to hold the dataset. Interactions can happen over SSH or we could host a service on a web server. The main resources will be webpages, pdfs, and xml documents. To interface with the assistant ARGUS will have integrated speech recognition along with a conversation text based interface. Most output will be text based for alpha testing phases
